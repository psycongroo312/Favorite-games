# 🎮 Favorite Games

**Твоя личная игровая коллекция + ИИ-портрет**

Собирай любимые игры по категориям (Favorite Protagonist, All time Favorite, Best Boss, Favorite OST и т.д.), сохраняй выбор и получай психологический портрет от ИИ

---

### ⚠️ Важно
**Без VPN сайт не запускается**  



---

### 🌐 Попробовать!

**https://favoriteigry.netlify.app**

---

### ✨ Возможности

- Красивая сетка карточек в тёмном неоновом стиле
- Поиск игр через RAWG API + отображение обложек
- Сохранение всего в `localStorage` (данные остаются после перезагрузки)
- Кнопка **Share** — копирует выбранные карточки в виде списка
- Кнопка **«Психологический портрет от ИИ»**

---

### 🛠 Технологии

- **React** + TypeScript
- Vite
- Tailwind CSS
- Jest + React Testing Library
- Netlify Functions + Groq API
